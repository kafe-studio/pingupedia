# Sprint 001: Foundation & branding

**Status:** TODO
**Dashboard Sprint:** 3
**Cíl:** Očistit kostru, nastavit brand, vytvořit species schema se zdroji a licencemi, zavést pravidla pro zobrazování obrázků bez ořezu.
**Runy:** 001–003

## Scope

Projekt startuje z kostra template. Tento sprint připraví stavební kameny pro encyklopedii:

- Odstraní nebo přestylizuje šablonové artefakty (blog kolekce, default site config, titulky).
- Nastaví vizuální identitu — barevné schéma (ledová/mořská paleta), typografie čitelná pro děti, logo/wordmark.
- Zavede **globální obrázková pravidla** — utility třídy a komponenty, které zaručí, že obrázky nikdy nejsou ořezané.
- Vytvoří content collection schema pro **druhy tučňáků** se zdroji (URL + název + typ zdroje), licencemi fotek (autor, licence, odkaz), taxonomií a distribucí.

Po dokončení sprintu má projekt: funkční homepage s brandem, layout, schema pro druhy a alespoň 1 testovací druh s ověřeným obsahem a zdroji.
